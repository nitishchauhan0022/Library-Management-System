# Library-Management-System
Library management system using c++ and postgresql.
### Requirements
* C++ Compiler - GNU g++/gcc
* Postgres Database
* libpqxx library
* ncurses library
### Build
``` g++ main.cpp -lpqxx -lpq -lncurses ```
